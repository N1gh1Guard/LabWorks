# Qt GUI для Laba2

Этот проект содержит графический интерфейс для работы с базой данных (HeavyDB), кэшем (LRUCache) и алфавитным указателем (AlphabetIndex).

## Требования

- Qt 5.15+ или Qt 6.x
- CMake 3.16+
- C++20 компилятор

## Сборка

```bash
cd qt
mkdir build
cd build
cmake ..
make
```

Или в Qt Creator:
1. Откройте `qt/CMakeLists.txt` как проект
2. Выберите комплект Qt
3. Нажмите "Build"

## Использование

После сборки запустите `Laba2GUI` (или `laba2gui`).

### Вкладка "Database & Cache"

- **Put**: Добавить/обновить пару ключ-значение в БД
- **Get**: Получить значение по ключу (сначала проверяется кэш, затем БД)
- **Erase**: Удалить ключ из БД и кэша
- **Clear DB**: Очистить всю базу данных
- **Clear Cache**: Очистить кэш
- **Refresh**: Обновить список ключей и статистику

### Вкладка "Alphabet Index"

- **Build Index**: Создать алфавитный указатель из введенного текста
- **Lookup**: Найти страницы, на которых встречается слово
- **Show Pages**: Показать все страницы с текстом
- **Show Pointer**: Показать алфавитный указатель (слово -> страницы)

## Структура

- `main_qt.cpp` - точка входа Qt приложения
- `main_window.h/cpp` - главное окно с двумя вкладками
- `CMakeLists.txt` - конфигурация сборки

Функции для работы с БД и индексом находятся в `../hpp/GuiFunctions.hpp`:
- `DatabaseManager` - управление БД и кэшем
- `AlphabetIndexManager` - управление алфавитным указателем
