# Конспект для подготовки к экзамену

Курс «Информатика (организация и поиск данных)».

Конспект разбит на 6 разделов в папке [Конспект/](Конспект/). Каждый раздел содержит подробное изложение с реализациями на C++.

| 1. Структуры данных | [1_Раздел.md](1_Раздел.md) |
| 2. Алгоритмы сортировки | [2_Раздел.md](2_Раздел.md) |

## Формат экзамена

- **Форма:** устная, по билетам
- **Билет:** 2 вопроса из разных разделов
- **Оценка за вопрос:** 0–5 баллов
- **Итог:** сумма за оба вопроса плюс оценки за разделы курса
