# Конспект для подготовки к экзамену

Курс «Информатика (организация и поиск данных)».

Конспект разбит на 6 разделов в папке [Конспект/](Конспект/). Каждый раздел содержит подробное изложение с реализациями на C++.

## Оглавление

| Раздел | Файл |
|--------|------|
| 1. Структуры данных | [Конспект/1_Раздел.md](Конспект/1_Раздел.md) |
| 2. Алгоритмы сортировки | [Конспект/2_Раздел.md](Конспект/2_Раздел.md) |
| 3. Ленивые вычисления | [Конспект/3_Раздел.md](Конспект/3_Раздел.md) |
| 4. Методы поиска | [Конспект/4_Раздел.md](Конспект/4_Раздел.md) |

| 6. Идиомы и приёмы ООП C++ | [Конспект/6_Раздел.md](Конспект/6_Раздел.md) |

## Формат экзамена

- **Форма:** устная, по билетам
- **Билет:** 2 вопроса из разных разделов
- **Оценка за вопрос:** 0–5 баллов
- **Итог:** сумма за оба вопроса плюс оценки за разделы курса
